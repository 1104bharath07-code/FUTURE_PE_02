# FUTURE_PE_02

# AI Content Marketing using UGC Ads

## Future Interns - Prompt Engineering Task 2

### Project Overview

This project demonstrates how AI can be used to generate high-converting User Generated Content (UGC) advertisements for digital marketing campaigns.

### Selected Product

GlowSkin Vitamin C Face Serum

### Objective

Create a reusable AI prompt framework capable of generating:

* UGC Ad Scripts
* Multiple Hooks
* CTAs
* Captions
* Platform-Specific Ad Variations

### Tools Used

* ChatGPT
* Prompt Engineering Techniques
* Marketing Frameworks
* UGC Content Strategies

### Skills Demonstrated

* AI Content Marketing
* UGC Ad Creation
* Conversion Copywriting
* Prompt Engineering
* Ad Psychology
* Platform Adaptation

### Deliverables

* 10 UGC Hooks
* 5 Complete Ad Scripts
* Multiple CTAs
* Social Media Captions
* Structured Prompt System
