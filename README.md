# AI Content Marketing using UGC Ads

## Future Interns – Prompt Engineering Task 2

### Project Overview

This project demonstrates how AI-powered prompt engineering can generate high-converting UGC (User Generated Content) advertisements for educational businesses.

### Selected Business

BrightMind Coaching Academy, Chennai

### Service

NEET Coaching Program

### Objective

Create a reusable AI prompt system capable of generating:

* UGC Hooks
* Ad Scripts
* CTAs
* Social Media Captions
* Platform-Specific Content

### Skills Demonstrated

* Prompt Engineering
* AI Content Marketing
* UGC Ad Creation
* Ad Psychology
* Conversion Copywriting
* Brand Voice Adaptation

### Tools Used

* ChatGPT
* Prompt Engineering Frameworks

### Deliverables

* 10 UGC Hooks
* 3 UGC Ad Scripts
* 10 CTAs
* 5 Instagram Captions
* Platform Adaptation Prompt
* Generated Outputs

### Prompt Structure

Role → Business Details → Task → Requirements → Output Format

This structure ensures consistent, reusable, and conversion-focused content generation for any business.
